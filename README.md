# cat-crosshair-compatibility
A resource pack for Minecraft Java to make the resource pack Cat Crosshair by MordorsElite (modrinth) and the mod Shield Crosshair Indicator by Bluebird (modrinth) to be compatible.

### Supported Minecraft Versions:
26.2 - 26.1.x


Credits for the original crosshair design go to MordorsElite, mod credits go to Bluebird, and credits for the indicator art go towards LKF3xd (me).
